Todo Local storage

A simple todo list with local storage (the data will be persistent).

I'm using angular-local-storage (can be installed via bower).

The code is the same as the previous step, by using :

  • $watch and $watchCollection (to observe changes as well on the input and on the todo list)
  • localStorageService.add() and localStorageService.get() which wil set and get values into th local storage (the service is init at bootstrap)

Try to add somethings and reload the page, even, quit the browser, your data will be persistent.
